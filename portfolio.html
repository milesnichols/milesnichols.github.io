Migraine Diary

Migraine Diary is a secure web app I created to help users track and understand their headache patterns without the usual friction of daily logging. I built it so users can safely record symptoms, review their history, and spot likely triggers in one place.

  • Secure public sign-up and login for protected user accounts.
  • Daily migraine reports with triggers, medications, and discomfort tracking.
  • Interactive history views and visualizations for filtering logs and spotting correlations.
  • Admin dashboard protected by role-based access control for secure oversight.

Autonomous Robot Navigation

This project was part of an event assistance robot concept, where our team built a microcontroller-driven robot that could autonomously navigate an unknown course and react to obstacles in real time. I helped lead a six-person team and focused on turning raw sensor input into reliable movement and debugging tools.

  • Implemented C-based control logic for navigation, movement, and obstacle avoidance.
  • Integrated ADC, ultrasonic ping sensors, and servo-driven scanning to interpret the robot's surroundings.
  • Built a Python GUI for live sensor visualization and faster debugging during testing.
  • Supported team coordination with task delegation, regular syncs, and iterative testing.
